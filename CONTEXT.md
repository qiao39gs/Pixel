# 像素拼豆图纸生成器

一个纯客户端的图片→拼豆图纸转换工具，支持颜色量化、手动编辑、导出 PNG/PDF。

## Language

**移动端**:
指手机竖屏（<640px）场景。平板及横屏不在本期改善范围内。
_Avoid_: 移动设备、手机端（前者太泛，后者排除了竖屏平板但本期也不涵盖）

**面板标签 (Tab)**:
移动端底部的四分区切换控件：参数 / 画布 / 色卡 / 项目。每个标签对应一个全屏面板，同一时间只显示一个。
_Avoid_: 页签、标签页

**编辑模式**:
画布的手动修整状态，启用画笔、橡皮擦、魔棒、取色等工具。与之相对的是预览模式（只读、可平移缩放）。
_Avoid_: 编辑态、edit mode

**拖拽模式**:
编辑模式下的一种临时子模式，单指拖拽 = 平移画布。激活时清空当前画笔/橡皮/魔棒选择；通过画布右下角悬浮按钮切换，按钮高亮表示激活中。退出后需重新选择编辑工具。
_Avoid_: 平移模式、手型模式

**平移 (Pan)**:
在画布溢出容器时移动可视区域。预览模式下单指拖拽即可平移；编辑模式下需切换到拖拽模式。
_Avoid_: 拖动、滚动

**还原视图**:
将画布的平移偏移归零、缩放回到自动适配值，恢复到初始居中状态。通过画布右上角半透明按钮触发。
_Avoid_: 重置视图、居中

**悬浮工具栏**:
编辑模式下画布右下角的可折叠竖排按钮组，包含橡皮擦 / 去杂色 / 魔棒 / 色板 / 拖拽模式。默认折叠为单图标，点击展开。
_Avoid_: 浮动菜单、工具抽屉
